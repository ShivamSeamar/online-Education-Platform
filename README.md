### Run Locally

To run **Youdemi** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/shivamseamar/youdemi.git
```

Windows:

```bash
git clone https://github.com/shivamseamar/youdemi.git
```
